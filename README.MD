# E+管理后台标准版MVVM 版

1. 基于 avalon 的当页面应用
2. 实现前后端分离


## 目录结构

* entry.js 路口文件
* components 目录放置 avalon 组件
* controller 分页路由应用页面
* global 存放静态文件 images，less，iconfont 等
* vendor 存放 js 依赖库

## 初始化项目

```
  git clone http://git-frontpay.cn/e-admin-mvvm.git
```

在`e-admin-mvvm`目录上执行命令提示符

```
npm i -d
```

## 开发

```
npm run dev-server
```

## 发布

```
npm run deploy
```
